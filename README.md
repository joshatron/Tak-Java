Tak
===


### Introduction

This project is a CLI based tak game. It features a server and an AI player.

### Running

To install:

    ./install

A folder for the cli will be created, along with for the ai trainer.

To run the cli:

    ./tak

To run the trainer:

    ./ai-train

To run the server:

    ./game-server

To clean up the project:

    ./clean
